# Hytale Name Availability Checker
Allows you to check if a name is available in they hytale game through endpoint verification. You must first enter your login information so please read through the code before entering any code to ensure you can trust what is happening

1. Add your Hytale account information at the top of the file
2. Get your web headers for your system and add those to the top of the file
3. Add the list of names to the bottom
4. >$ pip install requests
5. Run it
6. Progress
